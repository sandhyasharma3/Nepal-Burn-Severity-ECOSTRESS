# Nepal Burn Severity Analysis
This repository contains R scripts used to analyze burn severity across Nepal using Sentinel-2 dNBR, ECOSTRESS vegetation water stress metrics, topography, land cover, and spatial predictors.

## Scripts

Annual_RF.R → Annual ECOSTRESS model
- AmPm_RF.R → AM/PM ECOSTRESS model

## Notes

Large raster datasets are not included due to size limitations.

Users must provide their own data and update file paths accordingly.
